- 予想

コンソールに以下の出力がある

```
{ answer: 42 }
{ answer: 0 }
```

- 結果

予想通り(開発者ツールを開いた状態のタブで HTML を開いた)


- 開発者ツールを開いた状態のタブで HTML を開く場合

```
{ answer: 42 }
{ answer: 0 }
```

- HTML を開いた状態のタブで開発者ツールを開く場合
```
Object
Object
```
Objectを開くと
```
{ answer: 0 }
{ answer: 0 }
```

- 期待した結果を得るためには

```
{ answer: 42 }
{ answer: 0 }
```
の出力を期待する場合、コードを以下に変更する。

```html
<!DOCTYPE html>
<html>
  <body>
    <script>
      let life = { answer: 42 };
      console.log(JSON.parse(JSON.stringify(life))); // 最初のログ
      life.answer = 0
      console.log(life)
    </script>
  </body>
</html>

```
